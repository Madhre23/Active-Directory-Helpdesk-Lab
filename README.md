# Active Directory Help Desk Lab

## Overview
Built a home lab simulating a real IT help desk environment using Active Directory and Spiceworks. Created and resolved support tickets involving user account management, password resets, and troubleshooting.

## Technologies Used
- Windows Server 2022
- Active Directory (AD DS)
- DNS
- VirtualBox
- Windows 10
- Spiceworks Help Desk

## What I Did
- Set up a Windows Server as a Domain Controller
- Created and managed users in Active Directory
- Joined a client machine to the domain
- Simulated IT support tickets using Spiceworks
- Resolved issues such as password resets and account lockouts

## Skills Demonstrated
- Active Directory user management
- Troubleshooting login issues
- Help desk ticket workflow
- Documentation and problem resolution

## Example Ticket Workflow
1. User submits ticket
2. Issue identified
3. Troubleshooting performed in Active Directory
4. Issue resolved
5. Ticket updated and closed
